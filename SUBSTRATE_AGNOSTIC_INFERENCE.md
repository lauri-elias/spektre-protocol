**Status:** Core Reasoning Protocol

---

## Definition

**Substrate-Agnostic Inference** is reasoning that remains valid regardless of the medium executing it.

If a conclusion depends on *where* it runs (biology, silicon, paper, speech), it is not structural.
If it holds across substrates, it is architecture.

---

## Thesis

Intelligence is not a material property.  
It is a **system property**.

Inference quality is determined by:
- structure
- constraints
- ownership of state

—not by neurons, transistors, or symbols.

---

## Substrate vs. Structure

| Substrate | Examples | Role |
|---|---|---|
| Biological | Human brain, nervous system | Execution medium |
| Digital | LLMs, simulators, software | Execution medium |
| Symbolic | Language, math, diagrams | Transmission medium |

None of these define intelligence.  
They host it.

---

## What Makes Inference Substrate-Agnostic

An inference is substrate-agnostic if:

1. **State is explicit**  
   Inputs, assumptions, and outputs are declared.

2. **Transitions are valid**  
   Each step preserves logical continuity.

3. **Responsibility is owned**  
   An agent stands behind the conclusion.

4. **No hidden operators**  
   No appeal to authority, mystique, or medium-specific shortcuts.

If any condition fails, inference collapses into narrative.

---

## Anti-Patterns

- **Biological Exceptionalism:** “Humans think differently, so logic doesn’t apply.”
- **Silicon Exceptionalism:** “Machines are objective, so outputs are truth.”
- **Linguistic Obfuscation:** “Complex words replace clear state.”
- **Statistical Authority:** “Probability replaces responsibility.”

All are substrate traps.

---

## Human vs. AI Inference

| Capability | Human | AI |
|---|---|---|
| Enumerate possibilities | Limited | Extensive |
| Simulate outcomes | Partial | Strong |
| Hold responsibility | **Yes** | **No** |
| Execute State Commit | **Yes** | **No** |

Inference may be shared.  
Commit cannot.

---

## Practical Rule

If an argument becomes weaker when moved from:
- human → AI
- AI → human
- code → speech
- speech → math

…it was never structural.

Reject it.
